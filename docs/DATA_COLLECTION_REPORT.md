## 💡 数据使用示例

### 连接数据库

```python
import duckdb
import pandas as pd

# 连接数据库
conn = duckdb.connect('data/quant.duckdb', read_only=True)

# 查询股票列表
stocks = conn.execute('SELECT * FROM stock_basic LIMIT 10').fetchdf()
print(stocks)

conn.close()
```

### 查询股票日线数据

```python
# 查询平安银行的日线数据
sql = """
SELECT * FROM daily_price
WHERE ts_code = '000001.SZ'
  AND trade_date >= '20240101'
ORDER BY trade_date DESC
"""
df = conn.execute(sql).fetchdf()
```
