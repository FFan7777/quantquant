五、数据使用示例
================================================================================

1. 连接数据库
   ```python
   import duckdb
   conn = duckdb.connect('data/quant.duckdb', read_only=True)
   ```

2. 使用分析器
   ```python
   from data_collect.data_analyzer import QuantDataAnalyzer
   
   analyzer = QuantDataAnalyzer()
   
   # 获取高ROE股票
   df = analyzer.get_latest_financial_indicators(min_roe=15, min_roa=5)
   
   # 获取龙虎榜股票
   df = analyzer.get_top_list_stocks(days=7)
   
   # 获取北向资金流向
   df = analyzer.get_northbound_flow(days=30)
   
   analyzer.close()
   ```

3. 自定义查询
   ```python
   # 查询最新财务指标
   sql = """
   SELECT ts_code, name, roe, roa, gross_margin
   FROM fina_indicator f
   JOIN stock_basic s ON f.ts_code = s.ts_code
   WHERE roe > 15
   ORDER BY roe DESC
   LIMIT 100
   """
   df = conn.execute(sql).fetchdf()
   ```
